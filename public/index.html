    Conservative by design

    Transparent rules, then human judgment.

    1. Inactive seats: count the full plan cost only when active days are zero.
    2. Overlapping tools: count 50% of the cheapest active duplicate plan—not the whole subscription.
    3. Usage outliers: count 25% of usage above 1.5× the active-seat median, only after a high threshold is crossed.
    4. Savings cap: never estimate more savings than the observed monthly spend.
